# Discovery

This folder defines how `VR.app` should keep finding new repositories related to
VR utilities, overlays, diagnostics, bridges, runtimes, accessibility tools,
and experimental XR workflows.

## Files

- `discovery-sources.md`  
  Search streams, query patterns, and discovery channels for finding new
  relevant repositories.

- `local-source-cache-workflow.md`  
  Rules for cloning shortlisted repositories into a local-only source cache for
  code-level study.

- `intake-pipeline.md`  
  Step-by-step process for taking a newly found project from discovery to
  catalog entry and deeper study.

- `watchlist.md`  
  Ongoing discovery focus areas and the most important families to expand next.
