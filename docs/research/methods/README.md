# Methods Catalog

This folder extracts reusable `methods`, `patterns`, and `approaches` from the
tracked repositories.

The point is to answer a different question than the project registry:

- not only "which repos do we know?"
- but also "which ways of building VR utilities have already been observed and
  validated across the ecosystem?"

## Files

- `vr-utility-methods-catalog.md`  
  Canonical catalog of methods and implementation approaches that can be used
  when designing new VR tools inside `VR.app`.
