# Reuse Plans

This folder contains focused reuse-plan documents for individual repositories or
small clusters of related repositories.

These documents answer questions such as:

- what can be reused directly;
- what is only an architecture reference;
- what not to import into `VR-apps-lab`;
- what licensing or technical caveats matter.
