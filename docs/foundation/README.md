# Foundation Docs

This folder contains the stable, product-level foundation for `VR-apps-lab`.

## Files

- `repository-positioning.md`  
  Canonical public-facing description of `VR-apps-lab` as a knowledge repository and
  reusable VR development base.

- `current-operating-context.md`  
  Current project-state handoff and the minimum assumptions a fresh session
  should carry into the repo.

- `repository-positioning-backlog.md`  
  Backlog for repositioning the public repository and improving bilingual entry
  points.

- `platform-foundation.md`  
  Core purpose, design principles, architecture direction, and the filter for
  evaluating new utilities.

- `public-roadmap.md`  
  High-level roadmap for growing the repository as both a knowledge base and a
  foundation for VR utility tools.
